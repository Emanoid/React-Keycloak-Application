User Defined Network:
docker network create keycloak-network

Set up Database
docker run -d --name postgres-keycloak --net keycloak-network -e POSTGRES_DB=wikijstest -e POSTGRES_USER=wikijs -e POSTGRES_PASSWORD=wikijs@highstep postgres

Connect Keycloak to Database
docker run --name keycloak --net keycloak-network jboss/keycloak -e DB_ADDR=hs-psql1 -e DB_USER=wikijs -e DB_PASSWORD=wikijs@highstep
